Trasparenza

Come è distribuito ThisTinti.

Questa pagina spiega in modo semplice che cosa viene pubblicato, come si verifica una release e quali limiti restano.

Licenza

ThisTinti è distribuito con licenza Apache 2.0. La licenza permette di studiare, modificare e ridistribuire il software rispettando le condizioni indicate nei documenti allegati.

Codice sorgente

Il pacchetto sorgente della release viene pubblicato insieme all'installer. In questo modo chi deve fare una verifica tecnica può scaricarlo senza passare dalla pagina personale del manutentore.

Come verificare il file

Per ogni installer ufficiale viene pubblicato un checksum SHA-256. Il checksum serve a controllare che il file scaricato sia identico a quello prodotto dalla build ufficiale.

La Public Preview non è ancora firmata digitalmente. Per questo Windows o il browser possono mostrare un avviso di reputazione.

Che cosa non viene nascosto

  • responsabilità dell'organizzazione sulle personalizzazioni e sull'uso operativo;
  • necessità di test esterni prima di una beta pienamente validata.
